#Functional Requirements 
 
Customers should be able to register and log in securely to access their account. 
 
Customers should be able to browse the cafe's menu, including drinks and cakes, and select items to add to their order. 
 
Customers should be able to view and modify their order in a cart. 
 
Customers should be able to place the order and receive a confirmation 
 
Customers should be able to write feedback and ratings. 
 
Customers should be able to receive notifications on order status updates.
